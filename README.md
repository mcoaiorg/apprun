# Premium World Clock (Expo React Native)

A modern utility mobile app with:

- Live world clocks updating every second
- Unlimited clocks with drag-and-drop reorder
- Swipe-to-delete and animated list changes
- City search with major global cities
- Multiple alarms with repeat days and local notifications
- Theme and time-format settings

## Run

```bash
npm install
npm run start
```

## Notes

- Alarm triggering while app is closed is handled by Expo local notifications.
- Data is persisted locally with AsyncStorage.
